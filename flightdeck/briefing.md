# Briefing — Accessibility-Manager

## Conventions

每次产生文件改动都必须提交一次，不把多个独立批次积压到同一个提交。

提交信息格式为 `<gitmoji> [scope] <简短改动说明>`：

- `gitmoji` 必须选自 https://gitmoji.dev/；
- `scope` 使用 `[core]` 或 `[manager]`；
- Magisk 模块、脚本、构建与项目级文件使用 `[core]`；
- Android 管理应用的功能与界面使用 `[manager]`；
- 改动说明保持简短、明确。
